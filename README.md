# Clothing Brand Website

A modern and responsive clothing brand website built using HTML, CSS, and JavaScript.
This project focuses on delivering a smooth user experience with clean design and interactive features.

## Features

* Product search functionality
* Add to cart system
* Dark mode toggle
* Smooth and responsive UI design
* Fast and user-friendly experience

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Future Improvements

* Payment integration
* User authentication (login/signup)
* Order tracking system
* Product filtering and categories
